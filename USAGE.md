# Usage Guide for Dual Purpose Motion Light with IoT Integration

## Operation Modes
1. **Auto Mode:**
   - The light automatically turns on/off based on motion detection.
   - Sensitivity can be adjusted through the app.

2. **Manual Mode:**
   - Users can control the light manually via the mobile app.
   - Suitable for specific lighting needs.

3. **Scheduled Mode:**
   - Set specific times for the light to turn on/off.
   - Ideal for regular usage times.

## Control Methods
- **Mobile App Control:**
  - Download the "Motion Light Control" app.
  - Connect to the smart light via Wi-Fi.
  - Use the app to switch between operation modes.

- **Voice Control:**
  - Compatible with Amazon Alexa and Google Assistant.
  - Enable the skill or integration in the app settings.

## Status Monitoring
- Use the mobile app to view:
  - Current status (on/off).
  - Battery level (if applicable).
  - Motion detection status.

## Troubleshooting
- **Light Not Responding:**
  - Check Wi-Fi connection.
  - Restart the light by turning it off and on again.

- **Delayed Response:**
  - Ensure that the app permissions are granted.
  - Update the app to the latest version.

- **Motion Detection Issues:**
  - Adjust the sensitivity settings in the app.
  - Ensure the sensor is not obstructed by objects.
